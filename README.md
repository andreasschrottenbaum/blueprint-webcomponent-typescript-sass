# A lean setup for a WebComponent

### Requirements:

- [ ] [Node.js](https://nodejs.org)

### Used Technoligies:

- [x] TypeScript
- [x] SASS/SCSS
- [x] webpack

## Installation

```shell
npm install
```

## Start dev server

```shell
npm start
```

## Build WebComponent

```shell
npm run build
```

## Use WebComponent

(Path related to the `dist` directory)

```html
<body>
  <my-component></my-component>

  <script src="main.js" type="module"></script>
</body>
```
