# A lean setup for a WebComponent

### Used Technoligies:

- [x] TypeScript
- [x] SASS/SCSS
- [x] webpack

## Installation

```shell
npm i
```

## Start dev server

```shell
npm start
```

## Build WebComponent

```shell
npm run build
```

## Use WebComponent

```html
<body>
  <my-component></my-component>

  <script src="main.js" type="module"></script>
</body>
```
